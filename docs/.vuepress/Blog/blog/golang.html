# 第一个程序

package main

import "fmt"

func main() {
   fmt.Println("Hello, World!")
}

第一行,package main为声明main包;

第三行,import "fmt"为引入标准库;

第五至七行,为main方法,方法体为标准输出一句"Hello World!"。
